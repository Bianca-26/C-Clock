# interactive Dual Clock (C++)
Technology: C++, [WinBGIm]

Project Overview: Co-developed a desktop clock application featuring both digital and analog interfaces with real-time synchronization.

Key Features: Implemented multiple time zone support, an integrated calendar, and a custom alarm system with audio alerts.

UI Customization: Designed an intuitive interface allowing users to personalize the visual theme, including dynamic color changes for the clock face and hands.
